# Design Notes

Phase 0 design direction:

- calm,
- premium,
- visual,
- useful,
- focused,
- readable,
- work-surface oriented,
- quietly confident.

Avoid:

- fake OS concepts,
- cyberpunk/dashboard noise,
- AI marketing UI,
- decorative clutter,
- too many badges/colors,
- raw developer-first file views.

The UI should feel close to the dashboard UI kit references in density, polish,
and dashboard craft, while being adapted to this product's workspace-orientation
problem.

See `docs/design/references/dashboard-ui-kit/` for the PNG reference set.

See `docs/UI_SYSTEM.md` for the canonical agent-facing UI rules derived from
these references.
