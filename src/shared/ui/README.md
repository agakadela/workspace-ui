# shared/ui

Reserved for genuinely shared UI primitives.

Current primitives:

- `AppShell.tsx` - shared top navigation, product chrome, and dark workspace
  canvas wrapper for Phase 0 screens.
- `StatusChip.tsx` - shared icon + text status treatment for safety, review,
  private, info, and mock-only labels.
- `WorkspacePrimitives.tsx` - shared tray and surface-tab primitives for
  preview/composer surfaces and dense object metadata strips.
